Place edge mask/profile assets here.

Suggested structure:

- `assets/edges/parged/`
- `assets/edges/rough-brick/`
- `assets/edges/wirecut/`

These files are served by the web app via `/api/assets/...`.

Example:

- local file: `assets/edges/parged/1f8wl0-parged5.png`
- URL: `/api/assets/edges/parged/1f8wl0-parged5.png`
